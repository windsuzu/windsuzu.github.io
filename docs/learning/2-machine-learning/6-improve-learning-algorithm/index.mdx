這個章節將介紹如何設計一個機器學習系統，以及如何評估演算法的準確度、防止 overfitting 以及如何利用大量資料來提升演算法的效能。為了解釋這些概念，我們將以一個垃圾郵件分類器為例，詳細解說如何收集資料、撰寫 algorithm 以及如何處理偏斜資料。
