Model and Cost Function

Model 與 Cost Function 之間有著非常密切的關係，這是由於 Model 是用來預測的結果，而 Cost Function 則是用來計算 Model 預測結果的誤差。因此，Cost Function 將會用於最小化 Model 中出現的誤差，讓 Model 的預測結果越來越精確。

Model Representation​

在監督學習 (supervised learning) 中，我們會有一些訓練集 (training sets) 來學習預測，例如用房屋大小來預測價格。

每一筆資料可以用 (x, y) 來表示，而要表達特定某一行訓練示例時可以用 (xi, yi)，其中 m 代表訓練示例的數量 (Number of training examples)、x's 代表輸入變數 (Input variables / features)、y's 代表輸出變數 (Output variables / features)。

Cost Function​

我們知道 Hypothesis 的公式為

hθ(x)=θ0+θ1xh_\theta(x) = \theta_0 + \theta_1xhθ​(x)=θ0​+θ1​x

那我們要帶入什麼值給 θ0\theta_0θ0​ 和 θ1\theta_1θ1​，讓這個線性方程能夠最精準的預測 y 呢？這就是我們要找的 Cost function，這個 Cost function 會用來衡量 Hypothesis 的預測結果與實際結果的差異，也就是說，我們要找到一個 θ0\theta_0θ0​ 和 θ1\theta_1θ1​ 的組合，讓 Cost function 的值越小越好。

Idea : Choose θ0,θ1 so that hθ(x) is close to y for our training examples (x,y)\text{Idea : Choose } \theta_0, \theta_1 \text{ so that } h_\theta(x) \text{ is close to } y \text{ for our training examples } (x, y)Idea : Choose θ0​,θ1​ so that hθ​(x) is close to y for our training examples (x,y)

所以我們得到一個公式 (Cost function)，用於最小化 θ0\theta_0θ0​ 和 θ1\theta_1θ1​。 Cost function 所算出來的值越接近 0，代表越精確，其實就是在算每一個「預測的 y」和「真實的 y」的差異，平方過後並求平均值（又稱為"Squared error function"，或"Mean squared error"）。

Intuition I​

假設我們有一個非常完美的 Hypothesis (h) 被 Output 出來，這個 function 剛好是 linear equation 直線完全 fit 到所有的 training examples，所以他對應的 Cost function (J) 會是 0 才對。

備註

我們假設 θ0\theta_0θ0​ 是 0，只要看 θ1\theta_1θ1​ 就好，所以 J 會呈現一個 x, y plane。其中 x 軸代表 θ1\theta_1θ1​ 的值，而 y 軸代表 Cost function 的結果。

在上圖左，當 θ1\theta_1θ1​ 等於 1 時，hypothesis 完全符合所有的 training sets，因此他在 cost function 中得到 0 的結果。

接著上圖右，當 θ1\theta_1θ1​ = 0.5 時，Hypothesis 和每個 training sets 都有一些差距，此時他的 cost function 就等於 0.58，我們可以依此類推得出所有的 θ1\theta_1θ1​，得出整個 cost function 圖表。

因為當 θ1\theta_1θ1​ 等於 1 時，cost function 最接近（等於）0，代表誤差最小，所以我們應該要 output 一個 hypothesis，他的 θ1\theta_1θ1​ 為 1，θ0\theta_0θ0​ 為 0 為最佳解。

Hθ(x)=θ0+θ1×x=0+1×x\begin{aligned} H_\theta(x) &= \theta_0 + \theta_1 \times x \\ &= 0 + 1 \times x \end{aligned}Hθ​(x)​=θ0​+θ1​×x=0+1×x​

Intuition 2​

案例中我們把 θ0\theta_0θ0​ 拿掉，因此 cost function 呈現為二維圖表；若 θ0\theta_0θ0​ 不等於 0，cost function 就必須使用 3D 模型 (或 contour plot) 才可以視覺化。

備註

可以看到右下為 θ0\theta_0θ0​，左下為 θ1\theta_1θ1​，而模型最凹之處就是最小誤差的地方

在上圖中，當 θ0\theta_0θ0​ = 800 且 θ1\theta_1θ1​ = -0.15 時，不用看 cost function 就知道它與真實的 training sets 有誤差，而 cost function 的值與中心點也有段距離。

而當 θ0=250,θ1=0.12\theta_0 = 250, \theta_1 = 0.12θ0​=250,θ1​=0.12 時，我們可以算出他的 cost function 非常接近中心點了，也就是一個非常好的 hypothesis 了。

最後我們複習一下重點，也就是說要創建一個 learning algorithm，讓它自動去找出 cost function 的最低點，並且 output 出一個理想的 (Hypothesis) !
